# Selenium Jupiter Toys Automation

This repository contains a Selenium + TestNG automation framework for [Jupiter Toys](http://jupiter.cloud.planittesting.com) using **Page Object Model (POM)**.

## Features

- Java + Selenium WebDriver
- TestNG framework
- WebDriverManager for driver setup
- Page Object Model structure
- Tests for Contact form and Shop cart
- CI/CD ready via Jenkins
- Maven build management

## Run Locally

```bash
mvn clean test
